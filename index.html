With the help of my master's program, as well as independent study, I am regularly adding more skills to my toolkit. Here are some of the projects I have worked on independently or as part of my degree.

THE WITCHER NETWORKX ANALYSIS

A Networkx Analysis to study relationships and interactions on a dataset based on The Witcher, done on Python.

GAME OF THRONES REGRESSION ANALYSIS

A Python project done on a Game of Thrones dataset to determine what factors affect character deaths, with a logistic regression analysis done.

A data exploration project for COVID-19 deaths and vaccinations using SQL Server.

A project done in Python using movie data to find out what features affect revenue.
